## 🛠️ Active Builds

### Windows Desktop Tools (Rust)

* **[tactile-win](https://github.com/prueba-error/tactile-win)**: Keyboard-driven window tiling tool (inspired by GNOME's Tactile).
    - Interfaced directly with `Win32` hooks and handled lower-level window positioning lifecycle in Rust.
* **[win-glide](https://github.com/prueba-error/win-glide)**: Move/resize windows using arrow keys.
    - Implemented cursor simulation and shell APIs to avoid input lag.
* **[win-float](https://github.com/prueba-error/win-float)**: Modal HUD to pin windows always-on-top and control transparency.
    - Designed custom overlays and managed global hotkeys natively.

### Web Projects
* **[taura-birra-react](https://github.com/prueba-error/taura-birra-react)**: React landing page for a craft brewery in La Plata.
    - Commission project. Focused on matching client specifications and styling modular component layouts.
* **[eventos-dsw](https://github.com/prueba-error/eventos-dsw)**: Event billboard web app for Mar del Plata.
    - College project. Designed database routing and integrated local community event data.

### Automation & Micro-utilities
* **[autohotkey](https://github.com/prueba-error/autohotkey)**: Personal AHK v2.0 script collection.
    - Automating repetitive OS actions and key-remappings to speed up my desktop.
* **[one-click-logout](https://github.com/prueba-error/one-click-logout)**: Chrome extension to instantly logout from pre-defined websites.
    - Learned how Chrome WebExtension lifecycles isolate content script DOM access.
